## Algorithm
> [627. Swap Salary](https://leetcode.com/problems/swap-salary/description/)

* **What** 有一张收入表，修改每行记录的性别（即原来‘男’改成‘女’，‘女’改成‘男’）。

* **How** case when条件更新。

* **Key Codes**
```
update salary set sex = case when sex = 'm' then 'f' else 'm' end
```
